## Overview

This directory contains Python scripts for reproducing numerical results in:

- **Companion F/G/H:** Proton junction, mass difference, weak interactions

---

## Requirements

- **Python:** 3.9 or later (tested on 3.11)
- **Dependencies:** numpy, scipy, matplotlib

---

## Setup

```bash
# Create virtual environment
python3 -m venv venv
source venv/bin/activate  # On macOS/Linux
# venv\Scripts\activate   # On Windows

# Install dependencies
pip install -r requirements.txt
```

## Script Descriptions

### `common/`

| Script | Purpose | Reproduces |
|--------|---------|------------|
| `full5d_reduction.py` | 5D action reduction framework | Framework v2.0 §4, Phase-4 derivation program |
| `solve_electron_soliton_bvp_v5.py` | Nonlinear BVP solver for brane soliton | Companion F electron soliton verification |

### `companion_B_wkb/`

| Script | Purpose | Reproduces |
|--------|---------|------------|
| `gaussian_step9.py` | Exact Gaussian integral evaluation | Companion B Step 9, M(q) and V(q) integrals |
| `compute_Rdet_v2.py` | Determinant ratio R_det computation | Companion B §3 prefactor calculation |

### `paper3_lifetime/`

| Script | Purpose | Reproduces |
|--------|---------|------------|
| `neutron_wkb_sensitivity.py` | WKB verification gates | Paper 3 Table 2, sensitivity analysis |

## Running Scripts

### Run all scripts (verification)

```bash
./run_all.sh
```

### Run individual scripts

```bash
# Activate virtual environment first
source venv/bin/activate

# Common modules
python common/full5d_reduction.py
python common/solve_electron_soliton_bvp_v5.py

# Companion B calculations
python companion_B_wkb/gaussian_step9.py
python companion_B_wkb/compute_Rdet_v2.py

# Paper 3 lifetime verification
python paper3_lifetime/neutron_wkb_sensitivity.py
```
