## Implementation Notes

1. **Where to add `\cite{}`**:
   - In "3D Baseline (empirical)" boxes → cite CODATA/PDG
   - First mention of external theorem → cite canonical source
   - Framework 2.0 boxes can include citations in the "3D shadow" description

2. **Do NOT cite**:
   - Internal EDC constructs (postulates, definitions, OPR items)
   - Claims derived within EDC framework
   - Mathematical steps that are elementary

3. **Citation style**:
   - Use `\cite{key}` for single refs
   - Use `\cite{key1,key2}` for multiple refs
   - Biblatex numeric-comp style (sorted by appearance)
