**Framework v2.0**

## Description

This document presents the mathematical foundation for Elastic Diffusive Cosmology (EDC): a 5D brane-world framework where particle properties emerge from membrane geometry.

Key contents:
- 5D Einstein-Hilbert action with Israel junction conditions
- Effective action reduction to 4D
- Bulk-brane energy exchange formalism
- Epistemic tagging system ([Der], [Dc], [P], [BL], [Cal])
